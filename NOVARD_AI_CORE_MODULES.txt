MODULE 1: AUTHENTICATION & USER MANAGEMENT
--------------------------------------------------------------------------------

Description:
    Handles user authentication, registration, and profile management using 
    Firebase Google Sign-In. Manages user sessions and profile data.

Key Features:
    • Google OAuth 2.0 authentication via Firebase
    • User profile creation and management
    • User data persistence in MongoDB
    • Session management
    • Profile photo and display name updates

Technologies:
    • Firebase Authentication (Frontend)
    • Firebase Admin SDK (Backend)
    • MongoDB User Collection

API Endpoints:
    • POST /saveUser - Save/update user after authentication
    • GET /getUser/:email - Retrieve user by email
    • GET /getUserProfile - Get current user profile
    • POST /updateUserProfile - Update user information

Files Involved:
    • Frontend: Firebase.js, LoginButton.jsx, UserProfile.jsx
    • Backend: userController.js
    • Database: user.js (model)

MODULE 2: SKILLS PLANNING
--------------------------------------------------------------------------------

Description:
    AI-powered career skills analysis and recommendation system. Generates

    Groq AI.

Key Features:
    • Career-based skill recommendations
    • AI-generated skill roadmaps
    • Skill categorization (technical, soft skills, tools)
    • Learning resource suggestions
    • Progress tracking

Technologies:
    • Groq AI for skill generation
    • MongoDB for skill storage
    • React for UI display

API Endpoints:
    • POST /api/skills - Generate skills for a career
    • GET /api/skills/:careerId - Retrieve skills by career ID
    • GET /api/careerIds - Get all career IDs
    • DELETE /api/skills/delete/:careerId - Delete skills

Files Involved:
    • Frontend: Skills.jsx, SkillsInlineView.jsx
    • Backend: skillsController.js
    • Database: skills.js (model)

MODULE 11: COMMUNITY FORUM
--------------------------------------------------------------------------------

Description:
    Stack Overflow-style community forum where students can post questions, 
    get answers from peers, and receive AI-generated suggestions.

Key Features:
    • Issue/question posting
    • Community comments and responses
    • Upvote/downvote system for issues and comments
    • AI-generated response suggestions
    • Issue status management (open, solved, closed)
    • Category and tag-based organization
    • Search functionality
    • User reputation through votes

Technologies:
    • MongoDB for issues and comments
    • Groq AI for AI-generated responses

API Endpoints:
    • POST /api/forum/issues - Create issue
    • GET /api/forum/issues - Get all issues
    • GET /api/forum/issues/:issueId - Get issue details
    • GET /api/forum/issues/:issueId/comments - Get comments
    • POST /api/forum/comments - Add comment
    • PUT /api/forum/issues/:issueId/status - Update status
    • POST /api/forum/issues/:issueId/vote - Vote on issue
    • POST /api/forum/comments/:commentId/vote - Vote on comment
    • GET /api/forum/search - Search issues
    • POST /api/forum/ai-response - Get AI response
    • POST /api/forum/comments/:commentId/ai-response - Generate AI reply

Files Involved:
    • Frontend: Forum.jsx, ForumGrid.jsx, IssueList.jsx, IssueDetail.jsx, 
                IssueForm.jsx, IssueCard.jsx
    • Backend: forumController.js, forumAIController.js
    • Database: forumIssue.js, forumComment.js (models)
