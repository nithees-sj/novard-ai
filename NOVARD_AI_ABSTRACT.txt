
Novard-AI is a comprehensive, AI-powered career development and learning platform built on the MERN stack (MongoDB, Express.js, React 18+, Node.js) and integrated with the Groq AI inference engine, designed to unify career planning, adaptive skill building, intelligent content interaction, and peer collaboration into a single cohesive ecosystem. The platform comprises 11 core feature modules — including secure Google OAuth 2.0 Authentication via Firebase, an AI-Powered Skills Planning engine that generates categorized career roadmaps, an AI-Powered Projects Generator for portfolio-ready project ideas with tech stacks and implementation guidance, an AI-Powered Resume Builder producing ATS-friendly templates, a Skill Unlocker module delivering gamified day-by-day learning plans with pre-assessment quizzes and curated YouTube video recommendations, a Notes Management system enabling PDF upload with AI chat, summarization, and quiz generation powered by Langchain vector embeddings for semantic retrieval, a Video Library with AI-driven recommendations, a YouTube Video Summarizer for transcript extraction and interactive AI chat with video content, an Educational Video Module for processing uploaded learning materials, a Doubt Clearance system providing instant AI tutoring with follow-up conversations and practice quizzes, and a Community Forum offering Stack Overflow-style Q&A with peer responses, upvote/downvote voting, and AI-generated answer suggestions. Architected as a three-tier system with 60+ RESTful API endpoints, 17 backend controllers, 14 MongoDB collections, 35+ React components across 22 pages, and external integrations with Firebase Authentication, Groq AI API, YouTube Data API v3, and MongoDB Atlas, Novard-AI eliminates the fragmentation learners face by consolidating every dimension of professional growth — from career goal-setting and skill acquisition to document-driven study, video intelligence, doubt resolution, and community knowledge-sharing — into one intelligent, personalized platform.
